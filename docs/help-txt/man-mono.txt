yarn man (stasige ssg) (mono mode)

Show all commands and its help

    yarn man -ch 〈cmd〉

-   -c - clear console

-   -h - help description

-   〈cmd〉 - 〈empty〉 | man | copy | new | vars | srv | wrk | doc

-   yarn man

-   yarn man man

-   yarn man new

-   yarn man vars

-   yarn man srv

-   yarn man wrk
