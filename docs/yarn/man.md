### yarn man (stasige ssg)
Show all comands and its help
```shell
yarn man -ch 〈cmd〉
```

* `-c` - clear console
* `-h` - help description
*  `〈cmd〉` - 〈empty〉 | man | copy | new | vars | srv | wrk | doc 

* `yarn man`
* `yarn man man`
* `yarn man copy`
* `yarn man new`
* `yarn man vars`
* `yarn man srv`
* `yarn man wrk`
* `yarn man doc`
