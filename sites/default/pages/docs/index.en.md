## Start


Clone repo to locally dev 

```shell
git clone git@github.com:wunderwa/stamdic.git
cd stamdic
```

Copy default site template and init git repo to save  

```shell
# cp -R ./sites/default ./sites/my-site
./wrk -c my-site
git init 
```
